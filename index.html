The Learn Enough Story

Background

Learn Enough to be Dangerous is an outgrowth of the Ruby on Rails Tutorial and the Softcover publishing platform.

This page is being created as an exercise for the Learn Enough HTML to Be Dangerous tutorial, which teaches the basics of Hyper Text Markup Language, the universal language of the World Wide Web. Other related tutorials can be found at learnenough.com

This site is a near duplicate of the sample site created to support that tutorial, which I have only changed to include information about my experience in taking these tutorials, as well as my goals for learning these skills.

My Learning Goals

I started the Learn Enough tutorials at the recommendation of a friend. I work in a technological field, but I have never had to do coding or programming as part of my regular work. But, with the increasing availability of AI-driven tools, I realized it was time for me to upskill - at least enough so that I could use and (even more important!) evaluate outputs from AI tools

Additionally, I have several hobbies, and I have occasionally thought to myself "wouldn't it be awesome if I could make X, Y, or Z" kind of tool, app, or whatever to help me do some parts of those hobbies more efficiently?" And lo and behold!, the Learn Enough tutorials offer me the foundational skills and knowledge that I need to start exploring some of these projects!

Why Learn Enough and not Some Other Option?

Before starting with Learn Enough, I did explore some other tutorial options. There are a seemingly infinite number of tutorials and tools available online to learn these skills, and many of them are completely free! But inevitably a pattern started repeating:

-I'd start a course or tutorial
-Everything would go great! ... for a while
-Suddenly the tutorial is assuming that I have all kinds of background knowledge that I just don't have
-Or! The tutorial is uber technical, focusing on miniscule details and academic arcana
-And I start to feel like I'm learning lots of academic knowledge, but I'm not sure how useful this information will be day-to-day

And then my friend recommended that I check out learnenough.com.

What makes Learn Enough different is that they start off by assuming that you know nothing. Beginning with Learn Enough Command Line to Be Dangerous, they introduce you to absolutely basic computer knowledge. They help guide you through downloading the right programs. And they leave enough ambiguity to allow you to build your technical sopistication by seeking out your own answers to some questions and challenges.

Not only am I much closer to making the personal projects that I want to make. Not only am I increasing my knowledge for more effectiveness in my job. BUT ALSO I've learned more about how my own computer works and how to take advantage of it, I'm thinking and problem solving in new ways, and I'm having so much fun while I'm learning!

And here you are, reading my words on a website that I made, soup to nuts, myself! I'm using the command line, tracking crap on GitHub, and building a (very) simple website! I can't wait to see what I build next.
