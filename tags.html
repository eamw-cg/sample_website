Important HTML tags

This page is designed as a quick reference for some of the common tags covered in Learn Enough HTML to Be Dangerous. In the process of making it, we'll learn how to make HTML tables via table and related tags.

The tables below don't include all HTML tags, but they do list many of the most important ones.

Block Elements

Tag Name Purpose
h1–h6 headings include a heading, levels 1–6
p paragraph include a paragraph of text
table table include a table
tr table row include a row of data
th table header make a table header
td table data include a table data cell
header header semantic tag that defines introductory content
div division isolates sections of a document for better application of layout techniques
ol ordered list creates an ordered, or numbered, list
ul unordered list creates a non-ordered, or bulleted, list
blockquote block quote creates a block of text that can be formatted on its own

Inline Elements

Tag Name Purpose Example Result
em emphasized emphasize, or italicize, text <em>technical sopistication</em> technical sopistication
strong strong make strong, or bold, text <strong>at least a billion people</strong> at least a billion people
a anchor make a hyperlink <a href="https://learnenough.com/">Learn Enough</a> Learn Enough
img image include an image <img src="https://bit.ly/1MZAFuQ" alt="Michael Hartl">
code code syntax print in a monospace font to show source code <code>this is source code</code> this is source code
span span group and apply style or scripts to specific elements <span>I want to do something special with this text.</span> I want to do something special with this text.
li list item a row or item in a list, whether ordered or unordered <li>list element</li>

    Document Tags

    Tag Name Purpose
    !Doctype Document Type Indicates what type of document this is, such as html, css, or markdown.
    html HTML, or Hyper Text Markup Language Indicates that this document is written using html
    head Document Header Indicates that this is the header, or the section of the document that contains foundational information such as the title, and any meta information.
    body Document Body Indicates that this is the body section of the document, or the section that contains all content that gets displayed.
    title Title The title of the document.
    meta Meta Tag Specifies "meta" informationa bout the document, such as character sets to use.
