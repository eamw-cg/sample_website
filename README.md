#H1 README file for LE HTML tutorial website, sample_website

This is a sample website created as an exercise while completing the tutorial [*LearnEnough™ HTML to be Dangerous*](http://learnenough.com/html-tutorial). The [*LearnEnough™*](https://learnenough.com) series of tutorials are an excellent resource for any beginner. In addition to checking out the tutorials on the website, you can also [join the email list](http://learnenough.com/#email_list) and/or [follow @learnenough](http://twitter.com/learnenough) on Twitter.

After finishing *LearnEnough™ HTML to be Dangerous*, you will know enough HTML to be ==dangerous==.
